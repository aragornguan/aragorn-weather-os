# Aragorn Guan Weather OS

**Aragorn Guan Weather OS — a futuristic UI weather app with a real-time 3D globe, live city weather data, and cyberpunk HUD design.**

![Design Language](https://img.shields.io/badge/design-cyberpunk%20HUD-00d4ff?style=flat-square) ![Three.js](https://img.shields.io/badge/Three.js-r152-ff9d00?style=flat-square) ![Weather API](https://img.shields.io/badge/weather-Open--Meteo-00ff88?style=flat-square)

---

## Features

- **Real-time 3D rotating globe** — Three.js r152 with satellite texture and atmospheric glow shader
- **Live weather data** — 30+ world cities via Open-Meteo API (free, no key required), refreshed every 10 minutes
- **Cyberpunk HUD design** — dark `#050a14` background, cyan/amber/green accent palette, Orbitron + Share Tech Mono fonts
- **City markers** — animated cone markers + pulse rings on the globe with live temperature labels
- **Detailed city panels** — current conditions, satellite imagery (Bing Maps), humidity, wind, precipitation
- **Scrolling weather ticker** — live city ticker across the bottom

## Files

| File | Purpose |
|---|---|
| `weather-os.html` | Main single-file app (~90 KB, all CSS + JS inline) |
| `world.jpg` | 4000×2000 equirectangular satellite texture (1.4 MB) |
| `embed_texture.py` | One-time script: embeds `world.jpg` as base64 → `world-texture.js` |
| `fix-and-open.bat` | Double-click launcher: runs embed script then opens the app |
| `docs/SKILL.md` | Full project map and developer reference |

> **Note:** `world-texture.js` (generated, ~1.9 MB) is excluded from this repo — it can be regenerated by running `embed_texture.py`.

## Quick Start

### Option A — Local server (recommended, satellite texture works natively)
```bash
python -m http.server 7777 --directory .
```
Open: `http://localhost:7777/weather-os.html`

### Option B — Direct file open (double-click)
Run `fix-and-open.bat` first to embed the texture, then open `weather-os.html` directly.

## Design System

```css
--cyan:      #00d4ff   /* primary accent */
--amber:     #ff9d00   /* warnings / hot temps */
--green:     #00ff88   /* positive / cool */
--red:       #ff3355   /* alerts / extreme */
--bg-deep:   #050a14   /* background */
```

Fonts: **Orbitron** (HUD headers) · **Share Tech Mono** (data readouts) · **Rajdhani** (body text)
